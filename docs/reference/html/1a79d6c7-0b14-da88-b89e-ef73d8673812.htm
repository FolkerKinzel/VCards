      Remarks

      vCard properties whose data is encapsulated by RelationProperty objects are in particular the vCard 4.0 properties RELATED and MEMBER, the vCard 2.1 and vCard 3.0 property AGENT, as well as non-standard properties for specifying the name of the spouse (such as X-SPOUSE).
