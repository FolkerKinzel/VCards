      Property Value

      Boolean
      true if the instance contains information about the year, otherwise false.

      Remarks

      The vCard 4.0 specification allows to omit the year in a date but the .NET data types don't. This property gets the information whether the year component of an encapsulated DateOnly or DateTimeOffset value should be treated as irrelevant or not.
